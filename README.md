  - Fare hareketlerini kayıt eder

  - Manuel macro yazmak için kaydedilen macroyu açıp düzenleyebilirsiniz
  - Düzenleme sonrasında macro'da herhangi bir bozulma mevcut ise kontrol edebilirsiniz.
